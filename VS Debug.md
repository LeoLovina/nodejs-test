# VS Code Debug

## Toggle Auto Attch
- Ctrl+Shift+P
- ![debugging](/Documents/images/ToggleAuto.png)
After enabling Auto Attach, you'll need to restart your terminal. Then, the debugger should attach to your program within a second.
